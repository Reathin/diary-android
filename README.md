### 功能
纯粹的一个日记本应用
* -提醒：设置提醒时间，每天提醒写日记
* -隐私密码：密码保护，不用当心内容被别人看到（其实也并没什么卵用）
* -备份：将数据备份到云端
* -导出：将数据导出到sdcard（.txt/.db）
* -夜间模式
* -发现：把单条日记发布到广场

因为我一直有写日记的习惯，就简单的写了一个写日记的应用。

因为在多台pc上写的，所以结构比较乱。
